## 开发与部署

建议环境：Node.js 9.x 以上

安装依赖

```bash
npm i
```

### 开发环境

运行

```bash
npm run dev
```

### 生产环境

全局安装 pm2

```bash
npm i pm2 -g
```

运行

```bash
npm run prod
```

更新

```bash
npm run update
```

更多功能请参考 https://www.npmjs.com/package/pm2
